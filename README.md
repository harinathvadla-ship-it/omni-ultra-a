# My GitHub AI Agent 🤖

This repository contains a Python-based agent that automatically responds to issues using GitHub Actions.

## How it Works
1. **Trigger:** When a new issue is opened.
2. **Logic:** `bot.py` checks the issue content.
3. **Action:** The bot leaves an automated response.

## Files
- `bot.py`: The Python logic for the agent.
- `.github/workflows/agent.yml`: The automation schedule.
